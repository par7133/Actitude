您好，欢迎来到 Actitude！

Acttitude 是一款轻量且简单的内部软件，用于注册活动或支持。

Actitude 根据 GPLv3 许可证发布，按原样提供，我们对其误用不承担任何责任。

行动名称来自两个词之间的恶作剧：“积极”意味着我们积极地做事，而“态度”。

第一步，使用左侧面板密码和盐字段创建要插入配置文件中的哈希值。请记住手动设置盐值。

当您要使用有限的 Web 服务器或 phpfpm 用户在 PHP 进程上下文中运行 Actitude 时，您必须遵循一些简单的指令以获得最佳的首次设置：
  1. 检查网络应用专用路径中“数据”文件夹的权限；并在配置文件中设置其路径。
  2. 在数据路径中创建“.ACT_history”和“.ACT_captchahistory”文件并授予它们写入权限。
  3. 完成配置文件的相应设置，具体为：
    • 正确配置 APP_USE 和 APP_CONTEXT。
    • 根据需要配置 DISPLAY 属性。
    • 根据需要配置最大历史记录项数（默认值：1000）。
